Troubleshooting

Below is a collection of troubleshooting notes regarding the Orion BMS.

Problems Viewing Data in Utility
  • No cell voltages appear (Orion Jr. BMS)
  • Problems connecting to the Orion Jr. BMS unit via PC software
  • No cell voltages are showing in the utility
  • Not all cell voltages appear in live cell data
  • Error output is on, but no error codes are present
  • Problems connecting to the Orion BMS unit via PC software
Problems with CANBUS
  • Diagnosing CANBUS Communication Problems
Problems with Cells
  • Cells are not balancing, but should be
  • Replacing one or more cells
  • Internal resistance values are incorrect (by a significant amount)
  • High resistance cell
Problems with Charging and State of Charge
  • Diagnosing J1772 Charging Related Issues
  • CANBUS Elcon Charger isn’t charging, but should be (for standard Orion BMS)
  • Generic on/off charger isn’t charging, but should be (for standard Orion BMS)
  • Brusa charger isn’t charging but should be (for standard Orion BMS)
  • Eltek charger isn’t charging, but should be (for standard Orion BMS)
  • Charger turns off before battery pack reports full state of charge
  • Diagnosing sudden jumps in State of Charge (SOC)
  • Discharge or charge relay turns off too soon
  • Cell voltages drop after the charge completes
Problems with Relays
  • Discharge or charge relay turns off too soon
  • Relays don’t turn on
Problems with the Orion Jr. BMS
  • No cell voltages appear (Orion Jr. BMS)
  • Problems connecting to the Orion Jr. BMS unit via PC software
  • Orion Jr. BMS does not power up (LED stays off)
Problems with Voltages
  • No cell voltages appear (Orion Jr. BMS)
  • Pack voltage is reading a different voltage than measured by a voltmeter
  • Diagnosing sudden jumps in State of Charge (SOC)
  • No cell voltages are showing in the utility
  • Cell voltages drop after the charge completes
  • Not all cell voltages appear in live cell data
  • Cell voltages appear incorrect (when under load)
  • Cell voltages appear incorrect (not under load)
Trouble Codes
  • Fault Code Failsafe Behavior Chart
  • P0A0C – Highest Cell Voltage Too High Fault
  • P0A0E – Lowest Cell Voltage Too Low Fault
  • P0A10 – Pack Too Hot Fault
  • P0A0F – Cell ASIC Fault
  • P0ACA – Charge Interlock Fault
  • P0A0D – Cell Voltage Over 5 Volts
  • P0A04 – Open Wiring Fault (or “Open Cell Voltage Fault”)
  • P0A03 – Pack Voltage Mismatch Error
  • P0A80 – Weak Cell Fault
  • P0A0B – Internal Software Fault
  • P0A0A – Internal Heatsink Thermistor Fault
  • P0A09 – Internal Hardware Fault
  • P0A12 – Cell Balancing Stuck Off
  • P0AC0 – Current Sensor Fault
  • P0A1F – Internal Cell Communication Fault
  • P0AFA – Low Cell Voltage Fault
  • P0AA6 – High Voltage Isolation Fault
  • P0A01 – Pack Voltage Sensor Fault
  • P0A02 – Weak Pack Fault
  • P0A81 – Fan Monitor Fault
  • U0100 – External Communication Fault
  • P0560 – Redundant Power Supply Fault
  • P0A05 – Input Power Supply Fault
  • P0A06 – Charge Limit Enforcement Fault
  • P0A07 – Discharge Limit Enforcement Fault
  • P0A08 – Charger Safety Relay Fault
  • P0A9C – Battery Thermistor Fault
  • Error output is on, but no error codes are present
